# Instrukcja SELECT

Przykład 1:

Wyświetl całą zawartość tabeli z regionami geograficznymi (nazwa tabeli: REGIONS).

```
select *
from regions;
```

Rezultat:


| "REGION_ID" | "REGION_NAME" |
| 1 | "Europe" |
| 2 |"Americas" |
| 3 |"Asia" |
| 4 | "Middle East and Africa" |

| 0:0 | 1:0 |
| -- | -- |
| 0:2 | 1:2 |
